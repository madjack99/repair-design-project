# repair-design-project

Homework: repair-design-project. Difficulty Level 3.

## How to start the project:

1. Git clone repo to your local machine.
2. cd into to repo locally
3. Install VS extension "Live server"
4. Go to index.html
5. Right click on the code -> Open with live server
6. Scss already compiled and connected to index.html using 'live sass compiler' vs code extension.
